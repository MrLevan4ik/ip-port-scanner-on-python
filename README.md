# IP Port Scanner on Python

Сканер прокси с проверкой работоспособности, измерением скорости и гибкой настройкой.

## Возможности

- Проверяет SOCKS4, SOCKS5, HTTP, HTTPS прокси
- Рекурсивный поиск .txt файлов в указанной папке
- Измерение задержки (ms) и пропускной способности (KB/s)
- 6 форматов вывода: TXT, CSV, JSON
- Многопоточная проверка
- Логирование ошибок в файл
- Прогресс-бар, ретраи, фильтры, геолокация и многое другое

## Зависимости

```bash
pip install pysocks requests urllib3
```

## Быстрый старт

```bash
python main.py
```

## Все опции CLI

### Основные

| Флаг | Описание | По умолчанию |
|------|----------|--------------|
| `-d`, `--input-dir` | Папка со списками прокси | `proxies/` |
| `-o`, `--output-dir` | Папка для результатов | текущая |
| `-t`, `--timeout` | Таймаут соединения (сек) | `5` |
| `-w`, `--workers` | Количество потоков | `30` |
| `--test-url` | URL для проверки скорости | `http://httpbin.org/get` |

### Опциональные фичи

| Флаг | Описание |
|------|----------|
| `--progress` | Прогресс-бар вместо вывода каждого прокси |
| `--retry N` | Повторять проверку N раз при неудаче |
| `--min-speed MS` | Отбросить прокси медленнее MS мс |
| `--limit N` | Ограничить количество проверяемых прокси |
| `--parallel-files N` | Сканировать N файлов параллельно |
| `--validate-ip` | Проверять формат IP перед подключением |
| `--geo` | Определять страну прокси (бесплатный API) |
| `--fetch` | Скачать публичные прокси-листы перед сканированием |
| `--fetch-urls URL ...` | Доп. URLs для скачивания прокси-листов |
| `--watch SEC` | Повторять сканирование каждые N секунд |

### Логирование

| Флаг | Описание |
|------|----------|
| `--no-log-errors` | Отключить логирование ошибок в файл |

## Примеры

```bash
# Базовый запуск
python main.py

# Своя папка, быстрая проверка
python main.py -d my_proxies/ -t 3 -w 50

# Прогресс-бар + ретраи
python main.py --progress --retry 3

# Только быстрые прокси, макс. 200 штук
python main.py --min-speed 1000 --limit 200

# Параллельное сканирование файлов + геолокация
python main.py --parallel-files 3 --geo

# Скачать прокси-листы и проверить
python main.py --fetch

# Мониторинг: проверка каждые 5 минут
python main.py --watch 300 --progress

# Полный набор
python main.py -d proxies/ -o results/ -t 10 -w 100 \
  --progress --retry 3 --min-speed 500 --limit 500 \
  --geo --validate-ip
```

## Формат входных данных

```
http://ip:port
socks5://ip:port
ip:port:protocol
ip:port              # тестирование всех протоколов
```

Строки с `#` — комментарии, пропускаются.

## Результаты

| Файл | Описание |
|------|----------|
| `1_all_working_proxies.txt` | Все прокси, отсортированные по скорости |
| `2_proxies_for_reuse.txt` | Готовые адреса `protocol://ip:port` |
| `3_ips_only.txt` | Только `ip:port` |
| `4_*_proxies.txt` | Отдельные файлы по протоколам |
| `5_proxies_analysis.csv` | Таблица для Excel |
| `6_proxies.json` | JSON для программистов |

## Логирование ошибок

Ошибки записываются в `logs/error.log`. При большом количестве ошибок программа предложит открыть лог клавишей **Y**.

## Ctrl+C

При нажатии Ctrl+C сканер корректно завершает текущие задачи и сохраняет уже найденные прокси.
