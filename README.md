# IP Port Scanner on Python

Сканер прокси с проверкой работоспособности и измерением скорости.

## Возможности

- Проверяет SOCKS4, SOCKS5, HTTP, HTTPS прокси
- Рекурсивный поиск .txt файлов в указанной папке
- Измерение задержки (ms) и пропускной способности (KB/s)
- Множественные форматы вывода: TXT, CSV, JSON
- Многопоточная проверка
- Логирование ошибок в файл с возможностью быстрого открытия

## Зависимости

```bash
pip install pysocks requests urllib3
```

## Использование

```bash
python main.py [опции]
```

### Быстрый старт

```bash
python main.py                          # папка proxies/, 30 потоков, таймаут 5с
```

### Опции CLI

| Флаг | Описание | По умолчанию |
|------|----------|--------------|
| `-d`, `--input-dir` | Папка со списками прокси | `proxies/` |
| `-o`, `--output-dir` | Папка для результатов | текущая директория |
| `-t`, `--timeout` | Таймаут соединения (сек) | `5` |
| `-w`, `--workers` | Количество потоков | `30` |
| `--test-url` | URL для проверки скорости | `http://httpbin.org/get` |
| `--no-log-errors` | Отключить логирование ошибок | логирование вкл. |

### Примеры

```bash
# Своя папка с прокси
python main.py -d my_proxies/

# Быстрая проверка: таймаут 3с, 50 потоков
python main.py -t 3 -w 50

# Результаты в отдельную папку
python main.py -o results/

# Полный набор опций
python main.py -d proxies/ -o results/ -t 10 -w 100 --test-url http://ifconfig.me
```

## Формат входных данных

```
http://ip:port
socks5://ip:port
ip:port:protocol
ip:port              # тестирование всех протоколов
```

Строки, начинающиеся с `#`, считаются комментариями и пропускаются.

## Результаты

Создаётся 6 выходных файлов (путь задаётся через `-o`):

| Файл | Описание |
|------|----------|
| `1_all_working_proxies.txt` | Все рабочие прокси, отсортированные по скорости |
| `2_proxies_for_reuse.txt` | Готовые адреса в формате `protocol://ip:port` |
| `3_ips_only.txt` | Только `ip:port` без протокола |
| `4_*_proxies.txt` | Отдельные файлы по каждому протоколу |
| `5_proxies_analysis.csv` | Таблица для Excel |
| `6_proxies.json` | JSON для программного использования |

## Логирование ошибок

Ошибки (таймауты, обрывы соединения, невалидные прокси) записываются в `logs/error.log`. Если за сканирование произошло много ошибок, программа предложит открыть лог-файл одной клавишей.

Чтобы отключить логирование:

```bash
python main.py --no-log-errors
```
