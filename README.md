# IP Port Scanner on Python

Сканер прокси с проверкой работоспособности и измерением скорости.

## Возможности

- Проверяет SOCKS4, SOCKS5, HTTP, HTTPS прокси
- Рекурсивный поиск .txt файлов в папке `proxies/`
- Измерение задержки (ms) и пропускной способности (KB/s)
- Множественные форматы вывода: TXT, CSV, JSON
- Многопоточная проверка (по умолчанию 30 потоков)

## Зависимости

```bash
pip install pysocks requests urllib3
```

## Использование

1. Положите файлы со списком прокси в папку `proxies/`
2. Запустите:

```bash
python main.py
```

### Поддерживаемые форматы входных данных

```
http://ip:port
socks5://ip:port
ip:port:protocol
ip:port          # тестирование всех протоколов
```

## Результаты

Создаётся 6 выходных файлов:

| Файл | Описание |
|------|----------|
| `1_all_working_proxies.txt` | Все рабочие прокси, отсортированные по скорости |
| `2_proxies_for_reuse.txt` | Готовые адреса в формате `protocol://ip:port` |
| `3_ips_only.txt` | Только `ip:port` без протокола |
| `4_*_proxies.txt` | Отдельные файлы по каждому протоколу |
| `5_proxies_analysis.csv` | Таблица для Excel |
| `6_proxies.json` | JSON для программного использования |

## Настройки

Переменные в начале `main.py`:

```python
TIMEOUT = 5        # Таймаут соединения (сек)
MAX_WORKERS = 30   # Количество потоков
```
